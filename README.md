# מנהל תורנויות — צפון מטכל ועורף

## פריסה ב-Vercel

1. העלה תיקייה זו ל-GitHub (repo חדש)
2. כנס ל-vercel.com → New Project → בחר את ה-repo
3. Framework: Vite
4. לחץ Deploy

## שינויים עתידיים
- ערוך `src/constants.js` לשינוי עובדים/תורנויות
- ערוך `src/api.js` לשינוי ה-Google Script URL
- דחוף ל-GitHub → Vercel מתעדכן אוטומטית
